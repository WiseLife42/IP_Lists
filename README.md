# Ip_Lists
Here is a command that allows you to generate your own IP addresses in a given range.  
# Ressources
> * [Prips](http://manpages.ubuntu.com/manpages/bionic/man1/prips.1.html) 
# Install & Use
```
apt install prips
prips -h
```
